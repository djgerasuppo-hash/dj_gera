# Carpeta `static`

Esta carpeta contiene los activos locales de la web estática de DJ GER.

- `hero.svg` → imagen principal del hero.
- `photos/photo-1.svg` ... `photo-4.svg` → imágenes de prueba para la galería.
- `content.en.properties` → texto en inglés.
- `content.es.properties` → texto en español.
- `videos.properties` → lista de videos de YouTube.

Puedes reemplazar estos archivos por tu propio contenido y mantener las mismas rutas si quieres que funcione sin cambiar el HTML/CSS.
